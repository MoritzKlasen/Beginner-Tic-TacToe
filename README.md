# Beginner Tic-Tac-Toe Game

A simple and fun Tic-Tac-Toe game built with JavaScript Node.js and Express.

## Project Purpose

This project was created for three simple reasons:
- **I was bored** - needed something fun to work on
- **Learning JavaScript** - wanted to practice and improve my JavaScript skills
- **Just for fun** - building things is enjoyable!

## Features

- Clean and modern user interface with gradient design
- Two-player gameplay (Player X vs Player O)
- Score tracking with localStorage persistence
- Smooth animations and hover effects
- Responsive design for mobile and desktop
- Easy reset functionality

Made with ❤️ for learning and fun!
